🚗 Application de Covoiturage en Algérie
Cette application permet de gérer le covoiturage en Algérie, en facilitant la mise en relation entre conducteurs et passagers de manière simple et efficace.

📁 Installation
Assurez-vous d’utiliser un environnement local tel que XAMPP.

Placez le dossier du projet dans le répertoire htdocs de XAMPP.
👉 Renommez le dossier en: covoiturage.

🗃️ Base de Données
Créez une base de données nommée covoiturage_dz dans phpMyAdmin.

Ouvrez le fichier Create_DB_by_sql.sql situé dans le dossier db/.

Copiez le contenu de ce fichier dans l’onglet SQL de phpMyAdmin et exécutez-le pour créer les tables et insérer les données initiales.

📌 Une liste des utilisateurs par défaut est disponible dans le fichier db/user.txt.

🔐 Accès Administrateur
Pour accéder à l’espace administrateur, ouvrez votre navigateur et accédez à l’URL suivante :

 
http://localhost/covoiturage/admin/admin_login.php

Nom d’utilisateur : admin@admin.com

Mot de passe : admin
